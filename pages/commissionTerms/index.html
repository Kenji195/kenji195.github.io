Character(s) Pricing Rules

For the purposes of these pricing rules...
  • A Character is a unique identity or design (for example, an original character, a fictional character, a real person, an animal, or another identifiable subject).
  • A Character Display is an intentional, focal-point depiction of a Character.
    Note: The following do NOT count as separate Character Displays:
    • Decorative iconography, silhouettes, or insignias used as props or textures.
    • Character-shaped objects (e.g., a charm on a bracelet, a patterned fabric).
    • Ephemeral manifestations (e.g., magic energy, mist, or reflections) that lack significant facial or anatomical detail.
Pricing is based on the number of Character Displays, not the number of unique Characters. As a result, multiple depictions of the same Character (such as reflections, mirrored versions, character-shaped objects, or similar representations) are counted separately when they require individual drawing effort.

  1. MULTIPLE CHARACTER DISCOUNT
    1. When multiple Character Displays share the same style, only one (1) Character Display of that style is charged at the full base price.
    2. Each additional Character Display using that same style is charged at 50% (fifty percent) of that style's base price.
    3. If multiple styles are combined within the same commission, this pricing rule is applied independently to each style.
    4. Examples:
      • Three (3) Flat colored character displays = One (1) full price + Two (2) half price.
      • Two (2) Flat colored character displays + Two (2) Cell shaded character displays = One (1) flat price + One (1) half priced flat + One (1) full cell shaded + One (1) half priced cell shading
      • One (1) Lineless + One (1) Black shade = Both charged at full price.
  2. FRAMING
    1. Available framing options include Full Body and Head. Any Character Display depicting more than just the head is classified as Full Body for pricing purposes and is charged according to the Full Body price of the selected style.
    2. When a commission includes one or more Full Body Character Displays, one (1) Full Body Character Display is always charged at the full Full Body base price.
    3. This applies regardless of whether the Character Displays represent the same Character or different Characters, and regardless of how many Character Displays use each framing option.
    4. Any remaining Character Displays are then priced according to the Multiple Character Discount, using the corresponding framing and selected style for each Character Display.
